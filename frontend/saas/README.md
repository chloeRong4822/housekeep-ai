# 家政公司SaaS后台

## 技术栈
- Vue 3 + TypeScript
- Element Plus / Ant Design Vue
- Axios
- ECharts（数据可视化）

## 核心页面

### 1. 登录/注册
- 手机号验证码登录
- 公司信息完善

### 2. AI线索看板（首页）
- 顶部数据卡：今日新线索、本周成交、转化率
- 线索列表：AI评分、匹配标签、操作按钮
- 右侧地图：需求热力图

### 3. AI内容工厂
- 选择内容类型（阿姨展示/避坑指南/案例故事/促销）
- 填写参数（城市、服务类型、风格）
- AI生成3版内容
- 一键复制/导出

### 4. 阿姨管理
- 阿姨档案列表
- AI简历优化器
- 状态管理

### 5. 数据洞察
- 我的数据：线索量、转化率趋势
- 城市数据：供需比、薪资趋势
- AI建议

### 6. 账户中心
- 余额充值
- 消费记录
- VIP升级
